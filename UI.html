What are User Interfaces?

The user interface, in the industrial design field of humanâ€“computer interaction, is the space where interactions between humans and machines occur.

Command Line Interface

A command-line interface or command language interpreter, also known as command-line user interface, console user interface and character user interface, is a means of interacting with a computer program where the user issues commands to the program in the form of successive lines of text

Graphical User Interface

The graphical user interface is a form of user interface that allows users to interact with electronic devices through graphical icons and visual indicators such as secondary notation, instead of text-based user interfaces, typed command labels or text navigation

Menu Based Interface

This type of interface lets you interact with a computer or device by working your way through a series of screens or menus
